# Learn JQuery


## Install

- Para instalar o JQquery, ou seja, usa-lo há algumas formas:

	+ Pode-se apenas colcocar o endereço remoto do JQuery usando JSDelivr:
		`<script src="https://cdn.jsdelivr.net/npm/jquery@3.6.1/dist/jquery.min.js"></script>` 

	+ Usando npm ou yarn:
		`1 | npm install jquery`
		`1 | yarn add jquery`

	+ Ou mesmo, a mais comum que é baixando ele:
		- Vá no site: `https://jquery.com/`
		- Click em **Download**, instale a versão **uncompressed**
		- ! é recomendavel que se coloque o arquivo dentro da pasta `js`, criada dentro de seu projeto !
